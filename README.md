# homebrew-tap

Homebrew tap for [Sovia](https://sovia.space) — Desktop AI copilot for technical interviews.

## Links

- Website: https://sovia.space
- Main repository: https://github.com/rewolc/iinterview
- Releases repository: https://github.com/rewolc/sovia-releases

## Install

```bash
brew install --cask rewolc/tap/sovia
```

## Update

```bash
brew upgrade --cask sovia
```
